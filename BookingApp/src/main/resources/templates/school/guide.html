施設利用規約

KRCカラオケルーム 利用案内

  1. 利用時間を厳守：15時30分～17時30分の間に利用してください。
  2. 音量に注意：
    • 近隣の迷惑になるため音量は初期設定のままで使用し、ボリュームを上げないでください。
    • 近隣から苦情があった場合は利用停止となる可能性があります。
    • 必要以上に大声を出さないようにしましょう！
  3. ルーム内での喫煙・飲酒は禁止：
    • ソフトドリンク、食事はOKです。終了後は片付けをお願いします。
    • 違反が発覚した場合は次回以降利用停止となります。
  4. 機材は丁寧に扱うこと：
    • マイク・リモコン・モニターは精密機器です。乱暴に扱わないでください。
    • 機器破損の場合は弁償となります。
  5. 終了後は後片付けを行う：
    • 清潔な環境を保つために、きちんと後片付けをしてゴミは1階のゴミ箱に捨ててください。
    • 机・椅子等を移動した場合は原状復帰をお願いします。
  6. みんなで楽しみましょう：
    • 他の利用者への配慮を忘れず、快適な環境を維持しましょう。
    • ルールを守って楽しいカラオケタイムを過ごしましょう！

何かあれば、1～6号館管理部窓口までご連絡ください。

使用時
  1. ディスプレイのメインスイッチON
  2. ターミナル「LIVEDAM Ai」の左下ボタンを長押し（ランプ赤→緑に変わればOK）
  3. しばらくしてTVモニター、ターミナルとも起動します。
  4. ※ボリュームは初期設定から上げないようにお願いします。（近隣に迷惑をかけないため）
  5. キー（♭#）・テンポは変更可能です。
終了時
  1. 室内のゴミ回収、椅子・機器等を所定の場所に戻す。
  2. ターミナル「LIVEDAM Ai」の左下ボタンを長押し（ランプ緑→赤に変わればOK）
  3. しばらくしてターミナルがシャットダウンします。
  4. ディスプレイのメインスイッチOFF

KRCトレーニングジム 利用規約

学生の皆さんの健康増進と満足度向上を目的として、校内に『トレーニングジム』を設置しました。以下の利用規約を守り、快適で安全な環境をみんなで作りましょう。

  1. 利用対象者：
    本校の在籍学生
  2. 利用可能時間：
    平日 15:30 ～ 17:30
    ※左記以外の利用は禁止します
  3. 利用方法：
    予約不要です。空いている機材を自由にご利用いただけます。
    利用開始時・終了時に、入口に設置された使用者ボードに
    「クラス・氏名・使用時間帯・使用機材」を記入してください。
  4. 機材一覧：
    • トレッドミル（ランニングマシン）：5台
    • アップライトバイク（エアロバイク）：3台
  5. 使用時間の制限：
    一人が同一機材を使用できる時間は、最大60分までとします。
    複数人での譲り合いを心がけてください。
  6. 衛生管理：
    使用後は、備え付けのアルコールティッシュで汗などを拭き取ってください。
    次の人が気持ちよく使用できるよう、ご協力をお願いします。
  7. 飲食について：
    飲み物の持ち込みは可能です（水筒・ペットボトルなど）。
    食事（軽食・お菓子含む）は禁止とします。
  8. 禁止事項：
    • 機材の乱暴な扱い
    • トレーニング目的以外の使用
    • 裸足での利用（必ず運動用シューズを着用）
    • 他の利用者の迷惑となる行為
  9. その他：
    機材に異常があった場合は、速やかに職員まで報告してください。
    利用規約を守らない場合、利用を制限させていただくことがあります。

安全・清潔・譲り合いを大切に、みんなで快適なジム環境を作っていきましょう！

KRCブース 利用案内

ブースは主に就職活動の面接や面談用に使用する、一人用の防音室です。駅や公共施設にある面接ブースと同様の設備を備えています。

室内設備
  • 照明はフロント・サイド・トップの3か所にあり、それぞれ室内のつまみで光量調整が可能です。
  • 換気扇のオンオフスイッチも備えています。
利用ルール
  • 予約は30分単位で可能です。
  • 使用後は室内をきれいに整えてください。次に使う人のための配慮をお願いします。
  • 学生ホールでは休み時間などに多くの学生が過ごすため、ブース内にある「静寂のご協力をお願いします」という立札をホール内のテーブルに置くことを推奨しています。

快適かつ円滑に利用できるよう、ご協力をお願いいたします。
